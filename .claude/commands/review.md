# Review

Review and improve an existing document or piece of work in this workspace.

## Instructions

1. Read the target file(s) thoroughly.
2. Provide feedback on:
   - Clarity and structure
   - Completeness — are there gaps or missing considerations?
   - Accuracy — are there factual errors or questionable assumptions?
   - Actionability — can someone act on this?
3. Apply improvements directly to the file (with the user's agreement on major changes).
4. Commit the result when done.
