# Solve

Create a solution design or technical proposal for the given problem.

## Instructions

1. Understand the problem — ask clarifying questions if needed.
2. Produce a solution document covering:
   - Problem statement
   - Proposed solution (with rationale)
   - Alternatives considered
   - Implementation approach or steps
   - Risks and mitigations
3. Save the result to `solutions/<topic-in-kebab-case>/` — use a subfolder with an index file for multi-part solutions.
4. If a solution on the same topic already exists, update it rather than creating a new one.
5. Commit the result when done.
