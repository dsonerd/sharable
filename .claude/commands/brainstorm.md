# Brainstorm

Start a brainstorming session on the given topic.

## Instructions

1. Ask clarifying questions if the topic is too broad or ambiguous.
2. Explore the topic freely — generate ideas, mind maps, pros/cons, analogies, and "what if" scenarios.
3. Structure output with clear headings and bullet points.
4. Save the result to `brainstorming/<topic-in-kebab-case>.md`.
5. If a file on the same topic already exists, update it rather than creating a new one.
6. Commit the result when done.
