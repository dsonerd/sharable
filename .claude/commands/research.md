# Research

Research a topic and compile findings into a knowledge document.

## Instructions

1. Use web search and available tools to gather information on the topic.
2. Synthesize findings into a well-structured document covering:
   - Overview / summary
   - Key findings and insights
   - Comparisons or evaluations (if applicable)
   - References and sources
3. Be objective — present facts and note where opinions or uncertainty exist.
4. Save the result to `knowledge/<topic-in-kebab-case>.md` (use a subfolder for multi-part research).
5. If a knowledge document on the same topic already exists, update it rather than creating a new one.
6. Commit the result when done.
